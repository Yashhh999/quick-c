Introduction to C

C is a general-purpose, procedural programming language developed by Dennis Ritchie at Bell Labs in 1972. It is one of the most widely used programming languages and serves as the foundation for many modern languages like C++, Java, and Python.

Why Learn C?

  • Low-level access to memory and hardware
  • Efficient and fast execution
  • Portable across different platforms
  • Foundation for system programming, embedded systems, and operating systems
  • Rich standard library

Hello World Program

#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    return 0;
}

This simple program demonstrates the basic structure of a C program: including headers, the main function, and output.
