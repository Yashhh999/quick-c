Standard Library in C

The C standard library provides essential functions. Include appropriate headers.

<stdio.h> - Input/Output

  • printf, scanf - Formatted I/O
  • fopen, fclose - File operations
  • fread, fwrite - Binary I/O
  • getchar, putchar - Character I/O
  • fprintf, fscanf - File formatted I/O

Key Functions: printf uses format specifiers (%d, %f, %s). scanf can be tricky with input validation.

<stdlib.h> - General Utilities

  • malloc, free, calloc, realloc - Memory management
  • atoi, atof - String to number conversion
  • rand, srand - Random numbers
  • exit, abort - Program termination
  • qsort, bsearch - Sorting and searching

Advanced Note: qsort takes comparison function pointer for custom sorting.

<string.h> - String Handling

  • strlen, strcpy, strcat - Basic operations
  • strcmp, strncmp - Comparison
  • strchr, strstr - Searching
  • memcpy, memset - Memory operations

Many functions don't check bounds. Use strn* versions for safety.

<math.h> - Mathematical Functions

  • sin, cos, tan - Trigonometric
  • sqrt, pow - Power and root
  • ceil, floor - Rounding
  • exp, log - Exponential and log

Link with -lm on Unix systems.

<time.h> - Time Functions

  • time, clock - Current time
  • localtime, strftime - Formatting
  • difftime - Time difference

time_t is seconds since epoch. struct tm for broken-down time.

<ctype.h> - Character Classification

  • isalpha, isdigit, isspace - Classification
  • toupper, tolower - Case conversion
  • isalnum, ispunct - More checks

Locale-aware functions.

<assert.h>

assert(condition);  // Aborts if false in debug mode

Use for debugging preconditions. Disabled in release builds.

Other Headers

  • <limits.h> - Type limits (INT_MAX, etc.)
  • <float.h> - Floating point limits
  • <errno.h> - Error codes
  • <signal.h> - Signal handling

Advanced Note: Standard library is portable but basic. For advanced features, use external libraries.
