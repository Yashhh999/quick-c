File Handling in C

Files allow persistent data storage. Use FILE pointers for operations.

Opening a File

FILE *fp;
fp = fopen("file.txt", "r");  // Read mode
fp = fopen("file.txt", "w");  // Write (creates/truncates)
fp = fopen("file.txt", "a");  // Append

Other modes: "rb" binary read, "w+" read/write. Returns NULL on failure.

Reading from File

char ch;
while ((ch = fgetc(fp)) != EOF) {
    printf("%c", ch);
}

fgetc reads one character. EOF indicates end of file.

Writing to File

fprintf(fp, "Hello, %s!\n", name);

fprintf is like printf but to file. fputc for single characters.

Closing File

fclose(fp);

Always close files to flush buffers and free resources. Forgetting can cause data loss.

Binary Files

Store data as bytes, not text. Faster for large data.

fp = fopen("data.bin", "rb");
fwrite(&data, sizeof(data), 1, fp);
fread(&data, sizeof(data), 1, fp);

fwrite/fread handle structures, arrays directly.

Error Handling

if (fp == NULL) {
    perror("Error opening file");
    return 1;
}

Check fopen return. Use feof() and ferror() for read errors.

Random Access

fseek(fp, 0, SEEK_SET);  // Beginning
ftell(fp);  // Current position

Advanced Note: For large files or databases, consider buffered I/O and error recovery.
