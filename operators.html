Operators in C

Operators perform operations on operands. Understanding precedence and associativity prevents bugs.

Arithmetic Operators

Basic math operations. Division of integers truncates.

OperatorDescriptionExample
+Additiona + b
-Subtractiona - b
*Multiplicationa * b
/Divisiona / b
%Modulus (remainder)a % b

Advanced Note: Modulus with negative numbers can be implementation-defined. Use carefully.

Assignment Operators

Assign values to variables. Compound assignments are efficient.

int a = 5;  // Simple assignment
a += 3;     // Equivalent to a = a + 3
a -= 2;     // a = a - 2
a *= 4;     // a = a * 4
a /= 2;     // a = a / 2
a %= 3;     // a = a % 3

Comparison Operators

Return 1 (true) or 0 (false). Used in conditions.

OperatorDescriptionExample
==Equal toa == b
!=Not equal toa != b
<Less thana < b
>Greater thana > b
<=Less than or equala <= b
>=Greater than or equala >= b

Logical Operators

Short-circuit evaluation: right side not evaluated if left determines result.

  • && (AND): true if both true
  • || (OR): true if either true
  • ! (NOT): inverts boolean value

Bitwise Operators

Manipulate bits directly. Useful for low-level programming, flags, optimizations.

OperatorDescriptionExample
&Bitwise ANDa & b
|Bitwise ORa | b
^Bitwise XORa ^ b
~Bitwise NOT~a
<<Left shifta << 1
>>Right shifta >> 1
int a = 5;  // 101 in binary
int b = a << 1;  // 10, shifts left (multiply by 2)

Advanced Note: Shifting negative numbers or by large amounts is undefined. Use for performance-critical code.

Increment/Decrement

a++;  // Post-increment: use then increment
++a;  // Pre-increment: increment then use

Difference matters in expressions. Avoid complex expressions with side effects.

Operator Precedence

Multiplication before addition, etc. Use parentheses for clarity.
