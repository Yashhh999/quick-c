Pointers in C

Pointers store memory addresses. They enable direct memory manipulation, dynamic allocation, and efficient data structures.

Declaration

int *ptr;     // Pointer to int
char *str;    // Pointer to char
void *generic; // Generic pointer

The * indicates it's a pointer type. Pointers have their own size (usually 8 bytes on 64-bit).

Address Operator

int x = 10;
int *p = &x;  // p holds address of x

& gets the address of a variable. Pointers "point to" other variables.

Dereference Operator

printf("%d\n", *p);  // Prints 10
*p = 20;  // Changes x to 20

* accesses the value at the pointed address. Dereferencing null pointers crashes the program.

Pointer Arithmetic

Pointers can be incremented/decremented. Moves by the size of the pointed type.

int arr[5] = {1,2,3,4,5};
int *p = arr;  // Points to arr[0]
p++;  // Now points to arr[1]
printf("%d\n", *p);  // 2

Advanced Note: Pointer arithmetic is powerful for array traversal but dangerous if out of bounds.

Pointers and Functions

Pass pointers to modify arguments (pass by reference).

void swap(int *a, int *b) {
    int temp = *a;
    *a = *b;
    *b = temp;
}

int main() {
    int x = 1, y = 2;
    swap(&x, &y);  // Pass addresses
    // x is 2, y is 1
}

Without pointers, changes wouldn't affect originals.

Null Pointer

int *p = NULL;

Represents no valid address. Always check before dereferencing.

if (p != NULL) {
    *p = 5;
}

Double Pointers

int **pp;  // Pointer to pointer

Used for arrays of pointers, dynamic 2D arrays, etc.

Pointer to Functions

int (*func_ptr)(int, int) = &add;

Allows passing functions as arguments. Advanced topic for callbacks.

Advanced Note: Pointers are the source of C's power and complexity. Master them for systems programming, but avoid unnecessary complexity.
