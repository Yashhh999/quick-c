Control Structures in C

Control structures direct program flow. They execute code conditionally or repeatedly.

If-Else

Executes code based on a condition. Conditions evaluate to true (non-zero) or false (zero).

if (condition) {
    // Code if true
} else if (another_condition) {
    // Code if first false, second true
} else {
    // Code if all false
}

Key Points: Parentheses around condition are required. Use braces even for single statements to avoid bugs.

Switch

Efficient for multiple constant choices. Falls through unless broken.

switch (variable) {
    case 1:
        // Code for 1
        break;  // Exit switch
    case 2:
        // Code for 2
        break;
    default:
        // Code if no match
}

Advanced Note: Use switch for performance when many cases. Break prevents fall-through, which can be intentional but often a bug.

Loops

For Loop

Best for known iterations. Initialization, condition, increment in one line.

for (int i = 0; i < 10; i++) {
    printf("%d\n", i);
}

Variables declared in init are scoped to the loop (C99+).

While Loop

Executes while condition is true. Check condition before each iteration.

while (condition) {
    // Code
}

Do-While

Executes at least once, then checks condition.

do {
    // Code
} while (condition);

Use when loop must run once, like menu systems.

Break and Continue

break exits the innermost loop or switch. continue skips to next iteration.

for (int i = 0; i < 10; i++) {
    if (i == 5) continue;  // Skip 5
    if (i == 8) break;     // Stop at 8
    printf("%d ", i);
}

Advanced Note: Nested loops: break/continue affect only the inner loop. Use goto sparingly for complex control flow.
