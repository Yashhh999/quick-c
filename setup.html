Setting Up the C Development Environment

To start programming in C, you need a compiler and a text editor or IDE.

Compilers

  • GCC (GNU Compiler Collection): Free, open-source. Available on Linux, macOS, Windows (via MinGW or Cygwin).
  • Clang: Alternative to GCC, used in Xcode on macOS.
  • Visual Studio: On Windows, provides MSVC compiler.

Installation

On Linux (Ubuntu/Debian)

sudo apt update
sudo apt install gcc

On macOS

Install Xcode from App Store, or use Homebrew: brew install gcc

On Windows

Download MinGW from mingw-w64.org or use Visual Studio.

Compiling and Running

Save your code in a file with .c extension, e.g., hello.c

gcc hello.c -o hello
./hello

This compiles hello.c into an executable named hello and runs it.

IDEs and Editors

  • Visual Studio Code with C/C++ extension
  • Code::Blocks
  • Eclipse CDT
  • Vim/Emacs with plugins
