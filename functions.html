Functions in C

Functions modularize code, improve readability, and enable reuse. They must be declared before use.

Function Declaration

Specifies return type, name, and parameters. Body contains the logic.

return_type function_name(parameter_type parameter_name) {
    // Statements
    return value;  // If not void
}

Key Points: Return type can be any type or void. Parameters are passed by value (copies).

Example

int add(int a, int b) {
    return a + b;
}

int main() {
    int result = add(5, 3);
    printf("%d\n", result);
    return 0;
}

Prototypes

Declare function signature before definition. Allows calling before defining.

int add(int, int);  // Prototype

int add(int a, int b) {
    return a + b;
}

Prototypes are mandatory for functions defined after main.

Void Functions

Functions that don't return a value. Used for side effects.

void printHello() {
    printf("Hello\n");
}

Recursion

Function calls itself. Useful for problems with recursive structure, but watch stack overflow.

int factorial(int n) {
    if (n == 0) return 1;
    return n * factorial(n - 1);
}

Advanced Note: Recursion can be less efficient than iteration due to stack usage. Use for clarity when appropriate.

Pass by Reference

Use pointers to modify arguments.

void swap(int *a, int *b) {
    int temp = *a;
    *a = *b;
    *b = temp;
}

Advanced Note: Pointers enable efficient passing of large structures. Be careful with null pointers.
