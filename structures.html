Structures and Unions in C

Structures group related data into a single unit. Unions save memory by sharing space.

Structures

Define custom data types with multiple fields.

struct Person {
    char name[50];
    int age;
    float height;
};

Structure definition doesn't allocate memory; it's a blueprint.

Using Structures

struct Person p1;
strcpy(p1.name, "John");
p1.age = 30;
p1.height = 5.9;

Access members with dot operator. Initialize with {}.

Structure Pointers

struct Person *ptr = &p1;
printf("%s\n", ptr->name);  // Arrow operator

Use -> for pointers to structures. Equivalent to (*ptr).name.

Typedef

Create aliases for types to simplify code.

typedef struct Person Person;  // Now use Person instead of struct Person

Common for structures to avoid typing "struct" repeatedly.

Self-Referential Structures

struct Node {
    int data;
    struct Node *next;  // Points to another Node
};

Used for linked lists, trees. Forward declare if needed.

Unions

All members share the same memory location. Size is largest member.

union Data {
    int i;
    float f;
    char str[20];
};

Useful for variant types or memory optimization.

union Data d;
d.i = 10;  // Sets int
printf("%f\n", d.f);  // Undefined, different interpretation

Bit Fields

Pack data into bits for memory efficiency.

struct {
    unsigned int width : 4;  // 4 bits
    unsigned int height : 4;
} flags;

Advanced Note: Bit fields are non-portable. Use for hardware interfaces or tight memory constraints.
