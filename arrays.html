Arrays in C

Arrays store fixed-size sequences of elements of the same type. They provide contiguous memory access.

Declaration

Specify type, name, and size. Size must be constant.

int numbers[5];  // Uninitialized array
int arr[] = {1, 2, 3, 4, 5};  // Initialized, size inferred

Arrays are zero-indexed. Accessing beyond bounds causes undefined behavior.

Accessing Elements

numbers[0] = 10;  // First element
printf("%d\n", numbers[0]);

Use loops for processing entire arrays.

Multi-dimensional Arrays

Arrays of arrays. Useful for matrices, grids.

int matrix[3][3] = {
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9}
};

Access with matrix[row][col]. Memory is row-major order.

Array Size

Use sizeof to calculate length dynamically.

int size = sizeof(numbers) / sizeof(numbers[0]);

This works for arrays, not pointers.

Passing Arrays to Functions

Arrays decay to pointers. Pass size separately.

void printArray(int arr[], int size) {
    for (int i = 0; i < size; i++) {
        printf("%d ", arr[i]);
    }
}

Advanced Note: Functions can't determine array size from pointer. Always pass size to prevent buffer overflows.

Strings as Arrays

Strings are char arrays with null terminator.

char str[20] = "Hello";  // 'H','e','l','l','o','\0'

Advanced Note: String functions assume null termination. Buffer overflows are common security issues.
