Control Structures

for (int i=0; i<n; i++) { }
while (condition) { }

Functions

return_type func(params) { return val; }
void func(void) { }  // No params/return

Pointers

int *p;           // Pointer declaration
p = &x;            // Address of x
y = *p;            // Value at p
p++;               // Pointer arithmetic

Arrays

int arr[5];       // Fixed size
int arr[] = {1,2,3}; // Initialized
arr[0] = 10;       // Access

Strings

char str[20] = "Hello";
printf("%s", str);
strlen(str);       // Length
strcpy(dest, src); // Copy

Memory

int *p = malloc(sizeof(int));
free(p);

Compilation

gcc file.c -o output  # Compile
./output              # Run

Operators Precedence

  • Postfix: () [] -> . ++ --
  • Unary: + - ! ~ ++ -- (type)* & sizeof
  • Multiplicative: * / %
  • Additive: + -
  • Shift: << >>
  • Relational: < <= > >=
  • Equality: == !=
  • Bitwise AND: &
  • Bitwise XOR: ^
  • Bitwise OR: |
  • Logical AND: &&
  • Logical OR: ||
  • Conditional: ?:
  • Assignment: = += -= etc.
  • Comma: ,

Advanced Note: Use parentheses for complex expressions. Left-to-right for same precedence.
