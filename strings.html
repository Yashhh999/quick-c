Strings in C

Strings are arrays of characters terminated by null ('\0'). No built-in string type; use char arrays.

Declaration

char str[20] = "Hello";  // Fixed size
char *str2 = "World";    // Pointer to literal

String literals are read-only. Modifying them causes undefined behavior.

String Functions

Include <string.h> for standard functions.

  • strlen(str) - Length (excluding null)
  • strcpy(dest, src) - Copy src to dest
  • strcat(dest, src) - Append src to dest
  • strcmp(s1, s2) - Compare: 0 if equal

Key Points: strcpy doesn't check bounds. Use strncpy for safety.

Example

#include <stdio.h>
#include <string.h>

int main() {
    char name[50];
    printf("Enter name: ");
    scanf("%s", name);  // Reads until space
    printf("Hello, %s\n", name);
    return 0;
}

Input with Spaces

scanf stops at spaces. Use fgets for full lines.

char sentence[100];
fgets(sentence, 100, stdin);  // Reads entire line

fgets includes newline. Use strcspn to remove it.

String Manipulation

char str[20] = "Hello";
str[0] = 'J';  // "Jello"
strcat(str, " World");  // "Jello World"

Advanced Note: Always ensure destination has enough space. Buffer overflows are major security vulnerabilities.
